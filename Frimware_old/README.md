# ESP32 Stepper Motor Control

This is a simple PlatformIO project for ESP32 to control a 4-wire stepper motor (e.g. 28BYJ-48 with a ULN2003 driver).

## Pin configuration

- GPIO21 -> IN1
- GPIO22 -> IN2
- GPIO19 -> IN3
- GPIO18 -> IN4

## Build & Upload

1. Open this folder in VS Code.
2. Install PlatformIO extension if needed.
3. Build: `PlatformIO: Build`
4. Upload: `PlatformIO: Upload`
