# 电机与传感器模块说明

## 1. 模块设计原则

本模块采用模块化 C++ 实现，确保运动控制和传感采集职责分离，提高代码可读性与可维护性。

## 2. Motor 模块

### 2.1 功能

- 控制步进电机启停
- 设置速度与加速度
- 支持目标位置移动
- 提供运行调用接口

### 2.2 关键实现

- 使用 `AccelStepper` 库
- 采用 `DRIVER` 模式驱动外部步进驱动器
- 通过 `motor_run()` 持续执行运动

### 2.3 接口

- `motor_init()`
- `motor_set_speed(long speed)`
- `motor_move_to(long position)`
- `motor_run()`

## 3. Sensor 模块

### 3.1 功能

- 采集 HX711 力传感器输出
- 读取开/关限位开关状态
- 提供标准化数据接口

### 3.2 关键实现

- HX711 初始化、去皮和标定
- 使用 `INPUT_PULLUP` 配置限位开关
- 统一返回力值与限位布尔状态

### 3.3 接口

- `sensor_init()`
- `float sensor_get_force()`
- `bool sensor_is_open_limit()`
- `bool sensor_is_close_limit()`

## 4. 硬件模块协作

- 控制模块通过 `sensor_get_force()` 实现力反馈
- 限位状态用于保护闭环运动
- 步进电机位置控制与传感器状态共同构成夹爪运动闭环

## 5. 设计价值

- 支持后续加入 PID 力控算法
- 提供模块化接口，便于功能扩展与单元测试
- 使硬件驱动层与控制逻辑层解耦
