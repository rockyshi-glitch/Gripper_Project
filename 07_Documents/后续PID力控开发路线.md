# 后续 PID 力控开发路线

## 1. 目标概述

为实现稳定、精确的夹爪夹持力控制，后续开发将聚焦于基于力传感器反馈的 PID 控制算法，确保在不同对象和不同位移状态下实现恒定目标力。

## 2. 现有基础

- 已建立 `control` 模块框架
- 已实现力传感器读取接口
- 已支持步进电机位置控制与限位检测
- 已预留 `PIDParams` 与力控入口函数

## 3. 开发阶段

### 3.1 阶段 1：数据采集与标定

- 校准 HX711 传感器增益
- 获取力-电压映射并验证线性
- 明确夹爪空载和接触力区间

### 3.2 阶段 2：闭环控制架构设计

- 定义控制目标：夹持力值而非位置
- 选择模型：PI 或 PID 控制
- 设计控制周期与采样频率

### 3.3 阶段 3：PID 算法实现

- 在 `control.cpp` 内实现 PID 误差计算
- 使用 `target_force` 与 `sensor_get_force()` 生成控制指令
- 将 PID 输出映射为电机速度或目标位移增量

### 3.4 阶段 4：安全与限位保护

- 检查开/闭限位开关，避免机械撞击
- 当传感器读数异常时进入安全停机
- 添加失败检测与当前状态上报

### 3.5 阶段 5：调试与性能优化

- 调整 `Kp/Ki/Kd`
- 验证系统响应速度与稳定性
- 考虑抗积分饱和与误差微分滤波

## 4. 扩展建议

- 支持自适应 PID 参数整定
- 结合位置与力的复合控制策略
- 引入阻抗控制或力/位混合控制
- 添加远程诊断与参数在线调整接口
