# 系统总体架构说明

## 1. 项目背景

本项目旨在开发一套基于ESP32的力控自适应电动夹爪，适用于协作机器人末端执行器。系统目标包括实现可靠的夹持力控制、支持软夹持、提高重复定位精度，并优先采用低成本可制造硬件。

## 2. 系统目标

- 夹持力：5–40 N
- 夹持行程：30–50 mm
- 重复定位精度：< 1 mm
- 支持机械限位保护和力反馈控制
- 模块化软件架构，便于维护与后续扩展

## 3. 系统组成

### 3.1 硬件层

- ESP32 控制主板
- 步进电机驱动机构
- HX711 力传感器
- 开/关限位开关
- 夹爪结构与外壳

### 3.2 软件层

- Motor 模块：步进电机驱动与位置控制
- Sensor 模块：力传感器与限位开关数据采集
- Control 模块：夹爪控制逻辑与 PID 力控接口
- 主程序：系统初始化与运行循环

## 4. 关键功能

- 开闭夹爪运动控制
- 力传感器实时采集
- 限位开关安全保护
- PID 力控模块预留接口
- 模块化 C++ 实现，便于后续迭代

## 5. 系统优势

- 结构清晰，可维护性强
- 软硬件分层设计，便于调试
- 预留 PID 控制接口，为智能夹持力调节打基础
- 适用于作品集展示的工程化实现
